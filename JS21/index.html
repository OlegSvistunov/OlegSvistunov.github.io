Инструкция к HomeWork_JS_21

1. Скопировать репозиторий https://github.com/OlegSvistunov/OlegSvistunov.github.io/tree/master/JS21/

2. Открыть консоль в папке JS21
cd JS21

3. Установить зависимости
npm install jasmine --save-dev
npm install phantomjs-prebuilt --save-dev

4. Запустить тест
npm test

5. Выполнить функцию из консоли
node js\script.js <base> <power>
base - Основание степени
power - Степень
Например: node js\script.js 3 3
